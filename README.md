# Todo List with React
這是一個使用React打造的Todo List
# 功能
使用者可以使用新增、編輯、已完成或刪除todo
# 開始使用
1. 請先確認是否有安裝 Node.js 和 npm
2. 將此專案 clone 到本地
3. 終端機進入專案資料夾輸入：
```
npm install
```
4. 安裝完畢後，執行專案輸入：
```
npm start
```
5. 專案執行後在瀏覽器輸入： http://localhost:3000
6. 若要停止專案，在終端機輸入 ctrl + c
# 使用工具
- Node.js v14.16
- React @18.2.0
